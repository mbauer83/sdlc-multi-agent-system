# ADR-002: LangGraph for Orchestration Graph

## Context
The SDLC workflow requires stateful routing, phase gates, suspend/resume behavior for clarification, and deterministic transition control with specialist fan-out support.

## Decision
Use LangGraph for orchestration topology and stateful control flow.

## Rationale
- Native graph-state model matches phase and sprint lifecycle semantics.
- Conditional edge routing maps directly to gate evaluation and review control patterns.
- Supports interrupt/resume behavior needed for CQ and review interaction handling.
- Integrates cleanly with leaf-node agent execution via PydanticAI.

## Consequences
- Graph topology changes require explicit migration-aware updates.
- Version pinning and targeted regression testing are required for routing stability.
- Topology contracts must remain synchronized with framework/orchestration docs.
