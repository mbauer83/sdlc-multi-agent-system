## Algedonic Signal

A fast-path escalation record raised by any agent when a risk, safety violation, or cascading conflict is detected that requires immediate attention — bypassing normal hierarchical processing. Named after Stafford Beer's VSM algedonic channel. An Algedonic Signal carries a severity level (S1 highest), a trigger condition reference (ALG-nnn from the algedonic protocol), and the action required. All S1 signals halt normal processing immediately.

## Properties

| Attribute | Value |
|---|---|
| Lifecycle | Raised during BPR-002 (Skill Execution) or BPR-004 (Gate Evaluation) by any specialist or PM agent; processed by BPR-005 (Algedonic Escalation); resolved by PM or CSCO with user involvement for S1/S2 |
| VS Stage Coverage | May arise in any VS-001 or VS-002 stage; S1 signals halt the entire engagement until resolved |
| Safety Note | All algedonic signals are safety-relevant by definition |
| Application Layer Counterpart | DOB-005 (AlgedonicSignal) — carries signal id, severity, trigger-condition, raising-agent, phase, resolution-status |
