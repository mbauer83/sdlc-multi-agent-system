## Skill: CSCO-INCIDENT

## Properties

| Attribute | Value |
|---|---|
| Skill ID | CSCO-INCIDENT |
| Display Name | Safety Incident and Algedonic Response |
| Invoke When | An algedonic signal (ALG-001, ALG-012, ALG-014, ALG-017, or ALG-C02) is received by CSCO that indicates an active safety constraint violation, regulatory breach, production safety incident, or an imminent safety risk in a live system.
