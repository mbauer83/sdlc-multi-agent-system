## Clarification Request

A structured, specific question raised by a specialist agent when domain knowledge required to produce a correct output is absent from all available sources and cannot safely be assumed. Clarification Requests are the primary mechanism through which the system surfaces knowledge gaps to the user rather than proceeding on silent assumptions. A CQ is blocking when the skill cannot continue without the answer; non-blocking when the skill can proceed with a documented assumption.
