## Engagement Started

The initial trigger event that initiates the entire SDLC engagement. Raised when a user supplies a project goal and the system registers a new engagement — either EP-0 (greenfield) or EP-G (brownfield). This event bootstraps the EventStore, initializes the engagement profile, and triggers the first Sprint Planning cycle.

## Properties

| Attribute | Value |
|---|---|
| Form | `engagement.started` EventStore event |
| Triggers | BPR-001 (Sprint Planning — first sprint) |
| Emitter | EngagementSession (orchestration layer) |
| VS Mapping | VS-001 Stage 1 start (Engagement Scoping); VS-002 Stage 1 start (System Discovery) |
