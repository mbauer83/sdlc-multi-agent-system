## Brownfield Entry

The initiating event for VS-002 (Brownfield Onboarding). Raised when a user provides an existing target repository at entry point EP-G. This event triggers the Reverse Architecture process (BPR-008) rather than the standard Sprint Planning cycle, directing the system to begin with multi-source codebase scanning rather than Architecture Vision production.

## Properties

| Attribute | Value |
|---|---|
| Form | `engagement.started` EventStore event with `entry-point=EP-G` |
| Triggers | BPR-008 (Reverse Architecture — first pass) |
| Emitter | EngagementSession detecting EP-G entry point in engagement profile |
| VS Mapping | VS-002 Stage 1 start (System Discovery) |
