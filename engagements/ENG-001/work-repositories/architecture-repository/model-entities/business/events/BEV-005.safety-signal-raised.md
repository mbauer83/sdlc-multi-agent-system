## Safety Signal Raised

A high-priority event raised when any agent detects a condition matching an algedonic trigger (ALG-nnn) — such as a safety constraint violation, irresolvable conflict, or cascading risk. This event bypasses the normal sprint cycle and immediately activates the Algedonic Escalation process (BPR-005). S1 signals halt all active processing until resolved by the CSCO and PM with user involvement.

## Properties

| Attribute | Value |
|---|---|
| Form | `alg.raised` EventStore event |
| Triggers | BPR-005 (Algedonic Escalation) |
| Emitter | Any agent via `raise_algedonic` tool; orchestration layer for structural violations |
| VS Mapping | Can occur in any VS-001 or VS-002 stage; S1 signals halt the engagement |
| Safety Note | All instances safety-relevant |
