## Sprint Review Board

The structured collaboration between the PM Agent and the User (engagement owner) in the sprint review process. The PM Agent presents sprint output, the User evaluates it against expectations, submits decisions (approve / approve-with-corrections / re-run), and the PM processes those decisions before the sprint is formally closed. This collaboration is required — neither participant can complete the sprint review alone.
