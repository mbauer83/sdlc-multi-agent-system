## DE Agent

The Implementing Developer agent. Primary phase: G (code delivery). Implements work packages, submits pull requests, contributes test reports. Operates in a git worktree scoped to each sprint to prevent cross-sprint interference. Owns the delivery repository (metadata only — PRs, branch refs, test report references). Has read-write access to target repositories for code changes.
