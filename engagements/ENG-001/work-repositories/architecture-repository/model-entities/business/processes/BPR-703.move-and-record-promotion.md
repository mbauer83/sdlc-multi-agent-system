## Move and Record Promotion

The entity file is moved from the engagement `model-entities/` path to the enterprise repository under the correct layer subdirectory. An `artifact.promoted` EventStore event is recorded with the local ID, enterprise ID, and Architecture Board review reference. The engagement retains only the promotion event record.

## Properties

| Attribute | Value |
|---|---|
| Input | Updated entity file (post-sweep); enterprise repository target path; Architecture Board review reference |
| Output | Entity file at enterprise path; `artifact.promoted` event; engagement promotion log entry |
