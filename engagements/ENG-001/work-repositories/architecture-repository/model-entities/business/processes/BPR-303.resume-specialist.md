## Resume Specialist

The UserInteractionOrchestrator routes the `cq.answered` payload back to the suspended specialist agent node in the LangGraph graph. The specialist resumes skill execution at the point of suspension with the answer injected into working context. The CQ record (BOB-003) is marked resolved.

## Properties

| Attribute | Value |
|---|---|
| Input | `cq.answered` event with answer payload |
| Output | Specialist agent node resumed; BOB-003 marked resolved |
| Triggers | BPR-202 (Execute Skill Steps) — resumption of suspended execution |
