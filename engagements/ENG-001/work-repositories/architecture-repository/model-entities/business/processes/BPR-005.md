## Algedonic Escalation

A fast-path escalation that bypasses normal graph topology when a safety-critical or time-critical condition is detected. An `alg.raised` event routes immediately to the algedonic handler node, which escalates to the designated target (typically CSCO or User) outside the normal phase sequencing. Derived from Stafford Beer's VSM algedonic channel.

## Properties

| Attribute | Value |
|---|---|
| Trigger | `alg.raised` event from any specialist agent; severity S1 or S2 |
| Outcome | `algedonic.escalated` event; escalation target notified; workflow suspended pending resolution |
| Safety-Relevant | Yes |
