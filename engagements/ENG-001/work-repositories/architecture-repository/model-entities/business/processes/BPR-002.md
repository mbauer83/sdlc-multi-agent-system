## Skill Execution

A specialist agent is invoked with a specific skill. The agent assembles its 4-layer system prompt (identity + behavioral stance + skill instructions + tool results), reads relevant artifacts from its discovery scope, calls tools to produce outputs, and emits artifact events. The core inner loop of the system.

## Properties

| Attribute | Value |
|---|---|
| Trigger | `specialist.invoked` event emitted by PM agent tool |
| Outcome | One or more artifact files written; `artifact.created` / `artifact.updated` events recorded; optional `handoff.created` event |
| Safety-Relevant | No |
