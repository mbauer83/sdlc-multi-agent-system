## Select Specialist Invocations

Using the phase state assessment from BPR-101, the PM determines which specialist agents to invoke in this sprint, in which order, and with which work-package assignments. Considers dependencies between agents, outstanding handoffs, and gate obligations before composing the invocation list.

## Properties

| Attribute | Value |
|---|---|
| Input | Phase state assessment (from BPR-101); phase requirements; agent dependency map |
| Output | Ordered invocation list: [(agent, skill, work-package), ...] |
| Flow-to | BPR-103 (Dispatch Sprint) |
