## Enterprise Promotion

A matured engagement entity is promoted to the enterprise repository. The PromotionOrchestrator assigns a globally unique enterprise ID from `id-counters.yaml`, performs a deterministic reference sweep across all connection and diagram files (replacing the engagement-local ID with the enterprise ID), moves the entity file to the enterprise path, and emits an `artifact.promoted` event. Architecture Board review is required for the promotion to be accepted.

## Properties

| Attribute | Value |
|---|---|
| Trigger | User or PM initiates promotion for a baselined entity |
| Outcome | Entity file moved to enterprise repository; all references updated; `artifact.promoted` event recorded; Architecture Board review triggered |
| Safety-Relevant | No |
