## Reverse Architecture

EP-G entry point flow. Multi-source scan of target repositories (code, IaC, CI/CD, package manifests) and user-provided documents. Entity inference with `[inferred: <source>]` annotations. Batched user confirmation CQs. ERP model population. Gap and risk assessment. Hands off to CSCO for safety-relevant processes. Produces a reconstructed architecture model as the baseline for ongoing governance.

## Properties

| Attribute | Value |
|---|---|
| Trigger | User initiates engagement at EP-G with an existing target repository |
| Outcome | Populated model-entities/ corpus with inferred entities; gap assessment; `source.scanned` and `entity.confirmed` events recorded |
| Safety-Relevant | No |
