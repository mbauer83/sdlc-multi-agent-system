## Reference Sweep

The PromotionOrchestrator scans all connection files and diagram PUML files in the engagement repository for occurrences of the engagement-local entity ID and replaces each with the assigned enterprise ID. The sweep is deterministic and applied atomically across all affected files before the entity file is moved.

## Properties

| Attribute | Value |
|---|---|
| Input | Engagement-local ID; enterprise ID (from BPR-701); all connection and diagram files in engagement repository |
| Output | All connection and diagram files updated to reference enterprise ID; sweep log recorded |
| Flow-to | BPR-703 (Move Entity and Record Promotion) |
