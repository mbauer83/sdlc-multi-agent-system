## User Decisions Service

Bounded service that captures explicit user decisions used by governance and clarification workflows.

|---|---|
| Channels | CQ answers, file submissions, sprint-review decisions (approve/rework/pause/end/resume intent) |
| Consumers | BPR-003 (CQ Lifecycle) and BIA-001 (Sprint Review Interaction) |
