## Code Delivery Service

Provided by the DE Agent. Implements work packages assigned in Phase G, submits pull requests to target repositories via git worktrees, and contributes test reports. Produces delivery metadata records in the delivery repository. Has read-write access to target repository code branches.

## Properties

| Attribute | Value |
|---|---|
| Consumer | PM Agent (delivery tracking), QA Agent (test report consumption) |
| SLA | PR submitted within one sprint iteration per work package |
