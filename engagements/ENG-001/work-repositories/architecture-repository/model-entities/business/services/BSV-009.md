## User Decisions Service

The user's contribution as a bounded service consumed by the workflow. Includes: answering clarification requests (consumed by BPR-003), uploading supporting files (consumed by CQ lifecycle and reverse-architecture processes), and marking sprint review items (consumed by BPR-006). The dashboard is the user's interface to this service.

## Properties

| Attribute | Value |
|---|---|
| Consumer | PM Agent (CQ routing), UserInteractionOrchestrator (answer event detection), review_processing_node (review submission) |
| SLA | User-driven; no system SLA. Dashboard provides async notification of pending CQs and reviews. |
