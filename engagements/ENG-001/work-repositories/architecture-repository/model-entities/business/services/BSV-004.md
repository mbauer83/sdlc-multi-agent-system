## Safety Governance Service

Provided by the CSCO Agent. Provides gate vote authority on all phases (with veto power), produces safety constraint overlays and STAMP/STPA analyses, co-produces compliance assessments with QA, and handles incident response. Raises algedonic signals for safety-critical conditions detected in any phase output.

## Properties

| Attribute | Value |
|---|---|
| Consumer | PM Agent (gate evaluation), all specialist agents (safety constraints) |
| SLA | Gate vote within one sprint iteration; algedonic signal within current tool call |
