## Quality Assurance Service

Provided by the QA Agent. Produces test strategy and test plans (Phase E/F), executes tests against delivered code (Phase G), and co-produces compliance assessments with CSCO. Owns the QA repository. Has read-only access to target repositories.

## Properties

| Attribute | Value |
|---|---|
| Consumer | PM Agent (quality gate inputs), CSCO Agent (compliance co-production) |
| SLA | Test strategy produced in Phase E; test results within one sprint per delivery cycle |
