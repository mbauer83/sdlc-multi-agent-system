## Engagement Governance

The continuous organizational function responsible for sprint lifecycle management, phase gate evaluation, and overall delivery coordination. This function owns the rhythm of the engagement — deciding when to start, what to do, how to evaluate completion, and when to advance to the next phase. It is the meta-process that orchestrates all other functions.

## Properties

| Attribute | Value |
|---|---|
| Description | Continuous sprint-level governance: planning, gate evaluation, sprint review facilitation |
| Primary Processes | BPR-001 (Sprint Planning), BPR-004 (Gate Evaluation), BPR-006 (Sprint Review) |
| Primary Roles | ACT-002 (PM Agent — owner); ACT-001 (User — sprint review authority) |
| Realizes Capabilities | CAP-003 (Multi-Agent Orchestration) |
| Realizes Services | BSV-003 (Project Coordination Service), BSV-009 (User Decisions Service — via sprint review) |
| VS Stage Coverage | VS-001 Stage 1 (Engagement Scoping), all subsequent stages (gate between each) |
