## Knowledge Management

The organizational function responsible for capturing, retaining, synthesizing, and promoting architectural knowledge. This includes: recording agent learnings after each skill invocation; querying and applying prior learnings at the start of each skill; reverse-engineering architecture models from existing codebases (EP-G); and promoting engagement-level architecture assets to the enterprise repository for reuse across future engagements. This function creates the system's long-term institutional memory.
