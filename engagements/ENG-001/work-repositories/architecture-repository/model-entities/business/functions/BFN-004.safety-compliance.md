## Safety & Compliance

The continuous organizational function monitoring safety constraints and managing algedonic escalation across all phases. This function never waits to be invoked — it observes all agent activity and fires the algedonic channel immediately when a risk, safety violation, or cascading conflict is detected. It is the fast-path escalation mechanism that bypasses normal hierarchical processing when severity demands it.
