## Specialist Execution

The continuous organizational function through which all specialist agent roles apply their phase-specific skills to produce, validate, and hand off architecture and delivery artifacts. Every ADM phase artifact originates from this function. It is always invoked under PM direction (BFN-001) and runs to completion before returning control to the governance function.

## Properties

| Attribute | Value |
|---|---|
| Description | Skill-driven artifact production by all specialist agents across all ADM phases |
| Primary Processes | BPR-002 (Skill Execution) |
| Primary Roles | BCO-001 (Architecture Board — all agent roles collectively) |
| Realizes Capabilities | CAP-001 (Phase Execution), CAP-002 (Artifact Production) |
| Realizes Services | BSV-001 (Business Architecture), BSV-002 (App & Technology Architecture), BSV-005 (Requirements Management), BSV-006 (Code Delivery), BSV-007 (Quality Assurance), BSV-008 (Platform Engineering) |
| VS Stage Coverage | VS-001 Stages 2–5 (all artifact-producing stages) |
