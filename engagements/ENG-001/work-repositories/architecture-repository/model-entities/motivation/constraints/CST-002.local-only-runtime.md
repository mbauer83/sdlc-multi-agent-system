## Local-Only Runtime

No required external services beyond the LLM API. The engagement runtime (EventStore, ModelRegistry, LearningStore, dashboard) operates entirely on the local machine. No cloud database, no managed queue, no SaaS dependency. The only outbound call is to the Anthropic API.
