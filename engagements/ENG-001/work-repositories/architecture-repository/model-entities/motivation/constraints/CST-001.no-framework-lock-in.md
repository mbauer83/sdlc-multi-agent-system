## No Framework Lock-In

The LLM provider is swappable without changing any application or domain logic: provider selection is configured via `LLMConfig` (DOB-015) using PydanticAI's `provider:model-id` string convention. Changing from Anthropic to OpenAI or Ollama requires only a config change.

PydanticAI (agent execution) and LangGraph (orchestration control flow) are deliberate, permanent framework choices used for their respective capabilities — they are not subject to the swappability requirement. Domain logic (`src/models/`, `src/events/`) has zero framework imports.
