## No Framework Lock-In

The orchestration layer (LangGraph) and LLM client (PydanticAI / Anthropic API) must be swappable without changing domain logic. All I/O to frameworks is mediated through ports defined as `typing.Protocol` in `src/agents/protocols.py`. Domain logic has zero framework imports.
