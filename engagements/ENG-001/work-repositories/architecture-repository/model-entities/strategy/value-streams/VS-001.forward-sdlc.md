## Forward SDLC

The primary greenfield engagement flow: cold-start at EP-0, progressing through TOGAF ADM phases A through G in Agile ADM cadence, culminating in implemented and deployed software. Each phase produces baselined artifacts that gate progression to the next.

## Properties

| Attribute | Value |
|---|---|
| Trigger | User initiates engagement at EP-0 (no prior target system) |
| Outcome | Deployed software with complete architecture model, traceability chain, and governance record |
| Stages | Engagement Setup → Architecture Vision (A) → Business Architecture (B) → Information Systems (C) → Technology Architecture (D) → Opportunities & Solutions (E) → Migration Planning (F) → Implementation Governance (G) → Enterprise Promotion |
