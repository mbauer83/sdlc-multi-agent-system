## Brownfield Onboarding

The warm-start engagement flow for existing systems: entry at EP-G, multi-source codebase and IaC scan, entity inference with user confirmation, ERP model population, gap assessment, and transition into ongoing Phase G governance. Produces a reconstructed architecture model as the baseline for all subsequent work.

## Properties

| Attribute | Value |
|---|---|
| Trigger | User initiates engagement at EP-G with an existing target repository |
| Outcome | Reconstructed ERP v2.0 model (motivation/strategy/business/application/technology layers) with annotated gaps; system under Phase G governance |
| Stages | EP-G Entry → Codebase & IaC Scan (SA-REV-PRELIM-A) → Business Layer Reconstruction (SA-REV-BA) → Technology Layer Reconstruction (SWA-REV-TA) → Gap & Risk Assessment → User Confirmation → Phase G Governance |
