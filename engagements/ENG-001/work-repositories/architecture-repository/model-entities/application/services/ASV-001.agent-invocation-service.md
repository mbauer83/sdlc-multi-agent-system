## Agent Invocation Service

Named service grouping: APP-005 (AgentFactory) + APP-004 (SkillLoader) + APP-006 (AgentRegistry).

Accepts `(agent_id, skill_id, deps: AgentDeps)` → performs 4-layer prompt assembly → executes PydanticAI agent → returns structured result.

Execution sequence:
1. AgentRegistry lookup (APP-006) returns pre-built Agent instance
2. AgentFactory's `@agent.instructions` callback calls SkillLoader.load_instructions(skill_id) (Layer 3)
3. PydanticAI.run_sync(deps) executes the agent with full 4-layer context
4. Structured result (`str` or `PMDecision`) returned to LangGraph node

Governed by `framework/agent-runtime-spec.md`. Realises CAP-003 (Multi-Agent Orchestration capability).
