## Artifact I/O Service

Named service grouping: APP-002 (ModelRegistry) + AIF-004 (ArtifactReadWriterPort).

Provides read/write/list/search on ERP entity files with path constraints enforced per agent role. Keeps ModelRegistry in sync with the file system. Validates ERP format on every write. Emits EventStore events tool-transparently on every write.

Realises CAP-002 (Artifact Production capability) — every schema-valid, versioned, ERP-compliant artifact in the system is produced through this service.

## Properties

| Attribute | Value |
|---|---|
| Components | APP-002, AIF-004 |
| Realises | CAP-002 (Artifact Production) |
| Key invariant | ModelRegistry always consistent with file system after each write |
