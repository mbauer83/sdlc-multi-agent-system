## ReviewItem

`src/models/review.py`. Pydantic model for a single artifact decision in a sprint review. A list of `ReviewItem` objects forms the payload of the `review.submitted` EventStore event.

Fields:
| Field | Type | Description |
|---|---|---|
| `item_id` | `str` | UUID assigned at review creation |
| `artifact_id` | `str` | Artifact under review |
| `artifact_type` | `str` | Type (e.g. `business-architecture`) |
| `decision` | `Literal["approved","needs-revision","rejected"]` | User decision |
| `agent_tag` | `str \| None` | Agent to route revision to (if needs-revision) |
| `comment` | `str \| None` | User comment / revision instruction |

Review submission requires all items to have a non-null `decision` before `review.submitted` is accepted.

## Properties

| Attribute | Value |
|---|---|
| Module | `src/models/review.py` |
| Event payload | `review.submitted` carries `list[ReviewItem]` |
| Routing | `needs-revision` + `agent_tag` → `handoff.created` to tagged agent |
