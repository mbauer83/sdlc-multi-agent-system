## HandoffRecord

`src/models/handoff.py`. Pydantic model representing a structured inter-agent artifact handoff. Stored in `handoff-log/` markdown files and as an EventStore `handoff.created` event. Tracked in `WorkflowState.pending_handoffs`.

Fields:
| Field | Type | Description |
|---|---|---|
| `handoff_id` | `str` | `HND-NNN` assigned at creation |
| `from_agent` | `str` | Sending agent role |
| `to_agent` | `str` | Receiving agent role |
| `artifact_id` | `str` | Canonical artifact ID being transferred |
| `artifact_type` | `str` | Artifact type (e.g. `business-architecture`) |
| `version` | `str` | Artifact version at time of handoff |
| `handoff_type` | `str` | `"primary"` (first delivery) or `"revision"` (feedback loop) |
| `payload` | `dict` | Type-specific handoff payload (schema per artifact type) |
| `acknowledged` | `bool` | True once receiving agent has processed |
| `acknowledged_at` | `datetime \| None` | UTC timestamp of acknowledgement |

## Properties

| Attribute | Value |
|---|---|
| Module | `src/models/handoff.py` |
| Stored in | `handoff-log/` + EventStore `handoff.created` |
| Max iterations | 2 (initial + one revision) before PM escalation |
