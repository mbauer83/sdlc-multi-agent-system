## ArtifactRecord

`src/common/model_registry.py`. ModelRegistry's in-process representation of a scanned ERP entity file. Built from frontmatter at startup; never independently persisted — rebuilt by file scan on cold start or integrity mismatch.

Fields:
| Field | Type | Description |
|---|---|---|
| `artifact_id` | `str` | ERP artifact ID (e.g. `APP-001`) |
| `artifact_type` | `str` | Entity type (e.g. `application-component`) |
| `path` | `Path` | Absolute path to the entity file |
| `version` | `str` | Semver version string |
| `status` | `str` | `draft`, `approved`, etc. |
| `owner_agent` | `str` | Agent with write authority |
| `safety_relevant` | `bool` | From frontmatter |
| `produced_by_skill` | `str \| None` | Skill that produced this entity |
| `domain` | `str \| None` | Bounded-context domain tag |
| `engagement` | `str` | Engagement ID |
| `name` | `str` | Human-readable entity name |

Returned by `ModelRegistry.list_artifacts(**filter)`, `get_by_id(id)`, and as the first element of tuples from `search(query)`.

## Properties

| Attribute | Value |
|---|---|
| Module | `src/common/model_registry.py` |
| Lifetime | Ephemeral in-memory (not persisted; rebuilt from file scan) |
| Built from | ERP entity file frontmatter |
