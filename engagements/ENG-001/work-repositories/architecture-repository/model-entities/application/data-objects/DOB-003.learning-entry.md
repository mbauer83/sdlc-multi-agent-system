## LearningEntry

`src/models/learning.py`. Pydantic model for a single agent learning record. Validated by 9 schema rules before storage.

Fields:
| Field | Type | Description |
|---|---|---|
| `learning_id` | `str` | `<ROLE>-L-NNN` assigned by LearningStore |
| `agent` | `str` | Emitting agent role ID |
| `phase` | `str` | ADM phase in which the learning was generated |
| `artifact_type` | `str` | Primary output artifact type of the skill (for retrieval matching) |
| `trigger_event` | `str` | `"correction"` or `"synthesis"` |
| `error_type` | `str` | Error classification (e.g. `schema-violation`, `traceability-gap`) |
| `importance` | `Literal["S1","S2","S3"]` | S1 = critical; S3 = minor |
| `applicability` | `str` | Scope of applicability (e.g. `"all-engagements"`, `"this-engagement"`) |
| `correction_text` | `str` | The actionable correction content returned by `query_learnings` |
| `context_text` | `str \| None` | Additional context for the correction |
| `related` | `list[str]` | Graph links to related learning IDs (A-MEM pattern) |
| `synthesis_superseded` | `str \| None` | ID of learning this entry supersedes at synthesis time |
| `promoted` | `bool` | True if promoted to enterprise learning corpus |
| `generated_at_sprint` | `str` | Sprint ID at generation time |
| `generated_at_engagement` | `str` | Engagement ID at generation time |
