## LLMConfig

Pydantic model holding the configured LLM provider and model identifiers. Loaded at composition root via cascade: per-engagement YAML → global engagements-config.yaml llm: block → env vars (SDLC_PRIMARY_MODEL, SDLC_ROUTING_MODEL). Passed into build_agent() and EngagementSession. Uses PydanticAI provider:model-id string convention — changing provider requires only a config change. Realises AIF-002 (LLMClientPort).
