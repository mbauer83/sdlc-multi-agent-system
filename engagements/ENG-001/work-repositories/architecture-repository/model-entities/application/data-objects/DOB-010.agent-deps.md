## AgentDeps

**Type: `dataclass`, not Pydantic** — not serialised; constructed fresh at each LangGraph node invocation and discarded after the agent returns. All other DOB entities in this model are either Pydantic models (persisted via EventStore or files) or TypedDicts (DOB-012); AgentDeps is the one pure dataclass.

`src/agents/deps.py`. Injected by a LangGraph node into each PydanticAI agent invocation. Carries all context needed by the agent and its tools for one skill execution.

Fields:
| Field | Type | Description |
|---|---|---|
| `engagement` | `Engagement` | Full engagement config (DOB-002) |
| `event_store` | `EventStorePort` | Port to EventStore (AIF-001) for tool-emitted events |
| `active_skill_id` | `str` | Skill currently being executed (used by AIF-004 for tool-transparent emission) |
| `workflow_state` | `WorkflowState` | Current workflow state snapshot (DOB-009) |
| `engagement_base_path` | `Path` | `engagements/<id>/` base directory |
| `framework_path` | `Path` | `framework/` directory (for skill file loading) |

`AgentDeps` is a `dataclass` (not Pydantic) — it is constructed at the LangGraph node level, not serialised.

## Properties
