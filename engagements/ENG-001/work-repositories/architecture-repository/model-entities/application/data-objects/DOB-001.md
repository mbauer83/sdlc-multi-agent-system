## WorkflowEvent

`src/events/models.py`. Base Pydantic model for all EventStore events. All domain-specific event models inherit from this base.

Fields:
| Field | Type | Description |
|---|---|---|
| `event_id` | `str` | UUID assigned at creation |
| `event_type` | `str` | Dot-notation event type (e.g. `artifact.created`, `sprint.close`) |
| `agent` | `str \| None` | Emitting agent ID |
| `phase` | `str \| None` | Current ADM phase |
| `sprint_id` | `str \| None` | Current sprint ID |
| `skill_id` | `str \| None` | Active skill at time of emission |
| `payload` | `dict` | Event-specific data (structured per event type) |
| `sequence` | `int` | Monotonically increasing; never mutated; gap detection uses this |
| `timestamp` | `datetime` | UTC; set at record time |

The `payload` field carries event-type-specific data validated at the EventStore boundary.

## Properties

| Attribute | Value |
|---|---|
| Module | `src/events/models.py` |
| Storage | SQLite `events` table (append-only) |
| Parent | Base for all EventStore event types |
