## Engagement

`src/models/engagement.py`. Root domain entity. Loaded from `engagements-config.yaml` at session start and passed into `AgentDeps`. The aggregate root that all per-engagement data is scoped under.

Fields:
| Field | Type | Description |
|---|---|---|
| `engagement_id` | `str` | Unique engagement identifier (e.g. `ENG-001`) |
| `entry_point` | `str` | EP-0 through EP-H |
| `target_repositories` | `list[TargetRepo]` | Registered target repos (multi-repo support) |
| `primary_repo_id` | `str \| None` | ID of the `primary: true` repo; None if no target repo |
| `sprint_review` | `SprintReviewConfig` | Sprint review settings (enabled, timeout) |
| `uploads` | `UploadConfig` | Upload allow-list, max size, storage path |

`TargetRepo` sub-model: `id`, `label`, `role`, `domain`, `primary`, `url`, `local_clone_path`.

## Properties

| Attribute | Value |
|---|---|
| Module | `src/models/engagement.py` |
| Source | `engagements-config.yaml` |
| Role | Aggregate root; all workflow state scoped by engagement_id |
