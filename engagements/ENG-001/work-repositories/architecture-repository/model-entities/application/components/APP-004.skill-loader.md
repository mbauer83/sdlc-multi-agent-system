## SkillLoader

`src/agents/skill_loader.py`. Parses skill `.md` files and delivers the Layer 3 skill instructions for agent prompt assembly.

Responsibilities:
- Parse skill file YAML frontmatter + body
- Extract sections: Inputs Required, Steps, Algedonic Triggers, Feedback Loop, Outputs
- Enforce token budget by `complexity-class`: simple ≤ 600 tokens, standard ≤ 1 200 tokens, complex ≤ 2 000 tokens
- Truncation priority order: Algedonic Triggers → Feedback Loop → Outputs; Steps section is never truncated

Public API:
- `load_instructions(skill_id: str) → str` — returns assembled Layer 3 skill text within token budget

Skill resolution: maps `skill_id` to agent directory and filename via the agent index. Injected into AgentFactory as a dependency.

## Properties

| Attribute | Value |
|---|---|
| Module | `src/agents/skill_loader.py` |
| Consumers | AgentFactory (APP-005) — called during `@agent.instructions` callback |
| Token budget | 600 / 1200 / 2000 by complexity-class |
