## PM Agent

`src/agents/project_manager.py`. PydanticAI agent for the Project Manager role. Result type: `PMDecision`. VSM System 3 — operational management node in the LangGraph graph.

Tool set: `pm_tools` — `invoke_specialist`, `batch_cqs`, `evaluate_gate`, `record_decision`.

Loadable skills: all PM skills (master-agile-adm, phase-a, phase-e-f, phase-g, phase-h, retrospective-knowledge-capture).

The PM node is the primary routing decision-maker in the LangGraph graph. Returns a `PMDecision` with `next_action` field used by the routing function to select the next node.

Decision events emitted via PM tools: `specialist.invoked`, `gate.evaluated`, `cq.batched`, `decision.recorded`, `review.pending`.
