## MementoStore

Ephemeral continuity memory store. Wraps LangGraph BaseStore with OVERWRITE semantics in namespace (engagement_id, agent_role, "memento"), key = current_phase. Provides get_memento_state() and save_memento_state() tools to all specialist agents. Distinct from LearningStore (APP-003) in semantics (overwrite vs accumulate) and interface (AIF-007 vs AIF-006), but shares the same SQLiteStore backend (workflow.db).

## Notes

Implemented in src/agents/memento_store.py. Shares workflow.db with LearningStore and EventStore via LangGraph SQLiteStore. The BaseStore namespace separation ensures no key collisions. memento.saved EventStore events are informational — not replayed at session start (unlike learning.created). Governed by framework/learning-protocol.md §13.
