## AgentFactory

`src/agents/base.py`. Constructs a PydanticAI `Agent` instance for a given agent role. Implements the 4-layer system prompt assembly defined in `framework/agent-runtime-spec.md`.

Layer assembly:
- **Layer 1** — `system-prompt-identity` extracted from AGENT.md frontmatter (≤ 150 tokens; static)
- **Layer 2** — `### Runtime Behavioral Stance` from AGENT.md §11 (≤ 350 tokens; static)
- **Layer 3** — `SkillLoader.load_instructions(skill_id)` via `@agent.instructions` callback (injected per invocation by AgentDeps.active_skill_id)
- **Layer 4** — tool results appended to context at runtime by PydanticAI between tool calls

Public API:
- `build_agent(agent_id: str) → Agent[AgentDeps, str | PMDecision]` — reads AGENT.md, assembles 4-layer prompt, registers tool sets per agent role, returns configured Agent

Tool registration is role-specific: PM receives `pm_tools`; specialists receive `universal_tools` + `write_tools` (path-constrained) + role-specific extras.
