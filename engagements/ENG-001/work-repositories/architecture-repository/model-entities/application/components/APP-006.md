## AgentRegistry

`src/agents/__init__.py`. Module-level registry of pre-built PydanticAI Agent instances for all 9 roles. Populated at module import via `AgentFactory.build_agent()`.

Purpose: LangGraph nodes invoke agents without re-constructing them per invocation, avoiding repeated AGENT.md parsing and 4-layer prompt assembly overhead.

Structure: `AGENT_REGISTRY: dict[str, Agent]` keyed by `agent_id` string (e.g. `"project-manager"`, `"solution-architect"`).

Usage: `from src.agents import AGENT_REGISTRY; agent = AGENT_REGISTRY["software-architect"]`.

All 9 agent instances are eager-loaded at startup. If AgentFactory raises during construction (e.g. missing AGENT.md), the error surfaces at startup not at first invocation.

## Properties

| Attribute | Value |
|---|---|
| Module | `src/agents/__init__.py` |
| Contains | 9 Agent instances (PM, SA, SwA, DO, DE, QA, PO, SM, CSCO) |
| Consumer | LangGraph Orchestrator (APP-016) — nodes look up agents by role |
