## DashboardServer

`src/dashboard/server.py`. FastAPI application providing the local user interaction surface. Listens on 127.0.0.1 only (CST-002 — local-only runtime).

Features:
- All read-only view routes (artifact browser, sprint status, CQ list, review page)
- POST interaction routes (delegated to UserInputGateway — APP-021)
- SSE endpoint `/events/stream` — pushes EventStore events to browser in real time via `watchdog` observer; no WebSocket needed
- Mounts static files (minimal JS budget — two blocks only)
- Single process; no background workers

Technology choices rationale (ADR-004): FastAPI + Jinja2 server-side rendering; no build step; no frontend framework; SSE sufficient for push notifications; POST forms sufficient for CQ and review interaction.
