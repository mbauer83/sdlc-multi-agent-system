## LearningStore

`src/agents/learning_store.py`. LangGraph `BaseStore` wrapper providing per-role learning memory. Implements AIF-006 (LearningStorePort).

Namespace: `(engagement_id, agent_role, "learnings")`.

Public API:
- `query(phase, artifact_type, domain, expand_related) → list[str]` — metadata filter + graph expansion via `related` links (up to 2 extra results when primary filter < 3) + optional semantic supplement tier
- `record(entry: LearningEntry) → str` — validates schema (9 rules); assigns next `<ROLE>-L-NNN` id; writes to BaseStore AND writes `agents/<agent>/learnings/<id>.md` + updates `index.yaml`; emits `learning.created`

**Optional semantic tier:** gated on `sqlite-vec` availability; embedding model `sentence-transformers/all-MiniLM-L6-v2`; top-1 result above 0.75 cosine threshold supplements metadata filter when enterprise corpus ≥ 50 entries.

Cold-start rebuild: calls `rebuild_from_files()` if store is empty, rehydrating from `learnings/index.yaml`.

Optional `memory_20250818` backend: if configured, uses Anthropic's official memory tool type instead of direct SQLiteStore.
