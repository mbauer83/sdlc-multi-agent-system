## LearningStorePort

Port (typing.Protocol) for agent learning memory. Implemented by LearningStore (APP-003); swappable for in-memory fake in tests (no LangGraph BaseStore, no file I/O).

Method signatures:
- `query(phase: str, artifact_type: str, domain: str, expand_related: bool) → list[str]` — returns top-5 correction texts via metadata filter + graph expansion + optional semantic tier
- `record(entry: LearningEntry) → str` — validates entry schema (9 rules); assigns ID; persists; emits `learning.created`; returns assigned learning ID

All agents invoke this port via `query_learnings` and `record_learning` universal tools. The port is the clean boundary ensuring the domain (LearningEntry model) never imports LangGraph or file-system types.
