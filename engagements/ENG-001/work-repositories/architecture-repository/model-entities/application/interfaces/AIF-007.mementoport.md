## MementoPort

Interface contract for ephemeral memento state read/write. Exposes get_memento_state(phase) -> MementoState | None and save_memento_state(phase, key_decisions, open_threads, partial_context) -> None. All specialist agents call through this interface; implementation is MementoStore (APP-023). Analogous to LearningStorePort (AIF-006) for the LearningStore.
