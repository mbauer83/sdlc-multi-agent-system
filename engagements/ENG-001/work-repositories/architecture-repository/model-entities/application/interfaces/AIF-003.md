## SourceAdapterPort

Port (typing.Protocol) defined in `src/sources/base.py`. Uniform interface for all external source adapters (Layer 3 of the Discovery Protocol). Implemented by Confluence, Jira, external Git, UserUpload, and generic REST adapters in `src/sources/`.

Method signatures:
- `query(query: str) → str` — returns textual content relevant to the query
- `source_id: str` — unique identifier for this source instance (from `external-sources/<source-id>.config.yaml`)

All adapters are read-only — no write path from agents to external sources. Registered in the source adapter registry at session startup. Agents access external sources only via this port through `universal_tools.query_external_source(source_id, query)`.

## Properties

| Attribute | Value |
|---|---|
| Protocol location | `src/sources/base.py` |
| Implementations | Confluence, Jira, Git, UserUpload, REST adapters |
| Access mode | Read-only |
| Config | `external-sources/<source-id>.config.yaml` |
