## LLMClientPort

Port (typing.Protocol) defined in `src/agents/protocols.py`. The formal interface for the LLM completion layer. Implemented by PydanticAI `Agent` backed by the Anthropic API; swappable for a test double that returns deterministic structured output without LLM calls.

Method signature:
- `complete(messages: list, tools: list, result_type: type) → Result`

This port is what CST-001 (No Framework Lock-In) depends on for the LLM boundary. The domain and application layers never import `anthropic` SDK or PydanticAI types directly — they interact through this port at the composition root.

## Properties

| Attribute | Value |
|---|---|
| Protocol location | `src/agents/protocols.py` |
| Implemented by | PydanticAI Agent (Anthropic backend) |
| Test double | Deterministic stub (no LLM API calls) |
| Key constraint | CST-001 (No Framework Lock-In) |
