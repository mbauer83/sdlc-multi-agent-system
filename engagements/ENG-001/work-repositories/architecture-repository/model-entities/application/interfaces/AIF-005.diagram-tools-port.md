## DiagramToolsPort

Port (typing.Protocol) for diagram generation and validation tooling. Invokes the local `plantuml` CLI subprocess for rendering. Implementation lives in `src/agents/diagram_tools.py` (infrastructure layer — CLI subprocess is I/O).

Method signatures:
- `regenerate_macros(repo_path: Path) → None` — scans entity `§display ###archimate` blocks; regenerates `_macros.puml`
- `generate_er_content(entity_ids: list[str]) → str` — reads `§display ###er` blocks; returns PUML class declarations
- `generate_er_relations(connection_ids: list[str]) → str` — reads connection `§display ###er` blocks; returns cardinality lines
- `validate_diagram(puml_path: Path) → list[str]` — confirms all PUML aliases resolve to ModelRegistry entries; returns list of violations
- `render_diagram(puml_path: Path) → Path` — invokes `plantuml` CLI; returns path to rendered SVG (user-facing output only)

## Properties

| Attribute | Value |
|---|---|
| Protocol location | `src/agents/protocols.py` |
| Implementation | `src/agents/diagram_tools.py` (plantuml CLI subprocess) |
| Render output | SVG in `diagram-catalog/rendered/`; committed at sprint boundary |
