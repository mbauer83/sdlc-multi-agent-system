## EventStorePort

Port (typing.Protocol) defined in `src/agents/protocols.py`. The formal interface that all EventStore consumers depend on. Implemented by APP-001 (EventStore); swappable for an in-memory fake in tests.

Method signatures:
- `record_event(event: WorkflowEvent) → None`
- `replay() → WorkflowState`
- `replay_from_latest_snapshot() → tuple[WorkflowState, int]`
- `create_snapshot(trigger: str) → str`
- `check_snapshot_interval() → bool`
- `query_events(**filter) → list[WorkflowEvent]`

Injected into all agents via `AgentDeps.event_store` and directly into LangGraph nodes. CST-001 (No Framework Lock-In) depends on this port — the domain and application layers never import the concrete SQLite implementation.
