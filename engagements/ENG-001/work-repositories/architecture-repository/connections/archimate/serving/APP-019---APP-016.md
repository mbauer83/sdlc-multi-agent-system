PromotionOrchestrator (APP-019) provides enterprise promotion capability to the LangGraph Orchestrator (APP-016). When the PM node determines that a matured engagement entity is ready for promotion, APP-016 calls `PromotionOrchestrator.promote_entity()`, which handles the full promotion procedure: enterprise ID assignment, reference sweep, and file relocation. APP-016 waits for the result before emitting the `artifact.promoted` event.
