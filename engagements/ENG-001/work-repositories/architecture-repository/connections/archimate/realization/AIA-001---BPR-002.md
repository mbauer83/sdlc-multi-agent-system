Skill Execution (AIA-001) realizes the Skill Execution business process (BPR-002). This is a cross-layer realization from the application layer (ApplicationInteraction) to the business layer (BusinessProcess), which is valid in ArchiMate 3.1: application-layer behavioral elements can realize business-layer behavioral elements when they provide the implementation of that behavior. The interaction (AIA-001) — not the collaboration (ACO-001) or individual services — carries the realization, because realization is a behavioral relationship and must originate from a behavioral element.
